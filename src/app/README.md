# app

Next.js App Router directory containing the root layout, global styles, and the home page entry point.

## Files

| File | Description |
|------|-------------|
| `favicon.ico` | Browser tab icon for the application |
| `globals.css` | Global stylesheet importing Tailwind CSS and defining CSS custom properties for light/dark theme colors |
| `layout.tsx` | Root layout wrapping all pages; loads Geist Sans and Geist Mono fonts and applies base body styles |
| `page.tsx` | Home page component rendered at the `/` route |
